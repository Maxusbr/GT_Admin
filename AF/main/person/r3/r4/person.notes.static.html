Статическое описнаие
Сохранить


Можно использовать для описания уже добавленное. При этом произойдет поиск по тем описаниям, которые добавлены для персоны. Отображается только начало отписания. Найдя подходящее, выберете его из появившегося списка и нажмите ВВОД - описание будет скопировано в поле ниже. Внимание - копируется только текст описания, Вы НЕ редактируете предыдущее описание.

Статчиеское описание для персоны - это такое описание, которое описывается описанием описания. Тут Виталий должен какой то текст скостромить для пользователя, который подскажет ему, что это за хрень
Ключи (от 3 до 10)
